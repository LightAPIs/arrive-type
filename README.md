# arrive-type

A generic version of `@types/arrive`.

## Installation

```shell
npm install arrive
npm install @dragonish/arrive-type -D
```

## Usage

Add to the `tsconfig.json` file:

```json
{
	"compilerOptions": {
		"typeRoots": [
			"./node_modules/@types",
			"./node_modules/@dragonish/arrive-type"
		]
	}
}
```
